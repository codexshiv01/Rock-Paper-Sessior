# Contributing to Rock Paper Scissors Gesture App

Thank you for your interest in contributing! This document provides guidelines and instructions for contributing to this project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test thoroughly
6. Commit your changes: `git commit -m "Add: your feature description"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Create a Pull Request

## Development Guidelines

### Code Style
- Follow the existing code style
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused

### File Organization
- Place new components in `src/components/`
- Add utility functions to `src/utils/`
- Create new screens in `src/screens/`
- Update constants in `src/utils/constants.js`

### Testing
Before submitting a PR, ensure:
- The app runs without errors
- Camera functionality works
- Gesture detection is accurate
- UI is responsive on different screen sizes
- No console warnings or errors

### Commit Messages
Use clear, descriptive commit messages:
- `Add: [new feature]`
- `Fix: [bug description]`
- `Update: [what was changed]`
- `Refactor: [what was refactored]`
- `Docs: [documentation changes]`

## Areas for Contribution

### High Priority
- Improving gesture detection accuracy
- Performance optimization
- UI/UX enhancements
- Bug fixes

### Feature Ideas
- Multiplayer mode
- Additional gestures
- Sound effects
- Themes/customization
- Accessibility improvements

### Documentation
- Improve README
- Add code comments
- Create tutorials
- Write API documentation

## Pull Request Process

1. Update README.md if needed
2. Test on both iOS and Android if possible
3. Describe your changes in the PR description
4. Reference any related issues
5. Wait for review and address feedback

## Questions?

Feel free to open an issue for questions or discussions!

## Code of Conduct

Be respectful, inclusive, and constructive in all interactions.
